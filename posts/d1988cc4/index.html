windows 常用快捷键

更多阅读：

Windows 的键盘快捷方式

超全Windows10快捷键大全

  • Win + D：显示桌面
  • Win + E：打开资源管理器
  • Win + L：锁屏
  • Win + I：设置
  • Win + S：搜索
  • Win + V：多重剪贴板
  • Win + G：录屏
  • Win + R：打开运行窗口
  • Win + T：在任务栏上切换应用
  • Win + X：打开快速链接菜单
  • Win + Shift + S：截图
  • Win + Alt + D：显示日历
  • Win + [+ / -]：放大镜
  • Win + Esc：退出放大镜
  • Win + Tab：打开任务视图
  • Win + Ctrl + D：添加虚拟桌面
  • Win + Ctrl + F4：关闭当前虚拟桌面
  • Win + Ctrl + [← / →]：切换虚拟桌面

文件资源管理器

  • Ctrl + N：打开新窗口
  • Ctrl + W：关闭活动窗口
  • F11：最大/小化活动窗口
